## Summary

Anthropic launched a suite of ten pre-built agent templates targeting financial services workflows, available as plugins within Claude Cowork and Claude Code and as cookbooks for Claude Managed Agents. The release bundles research-focused agents (pitch builder, earnings reviewer, market researcher) with operations-focused agents (KYC screener, general ledger reconciler, month-end closer) alongside expanded data connectors and Microsoft 365 integrations. Claude Opus 4.7 leads Vals AI's Finance Agent benchmark at 64.37%, and major institutions including Citadel, BNY, and Mizuho are already in production.

## Key Highlights

- Ten agent templates covering research (pitch builder, earnings reviewer, market researcher, meeting preparer, model builder) and operations (valuation reviewer, GL reconciler, month-end closer, statement auditor, KYC screener)
- Microsoft 365 add-ins for Excel, PowerPoint, Word, and Outlook (coming soon) with cross-app context persistence
- Eight new data connectors: Dun & Bradstreet, Fiscal AI, Financial Modeling Prep, Guidepoint, IBISWorld, SS&C IntraLinks, Third Bridge, Verisk — plus Moody's MCP app covering 600M+ companies
- Each agent packages skills, connectors, and sub-agents; users retain review-and-approval control
- One reported outcome: AML investigations compressed from days to minutes

## Why It Matters

Anthropic is racing to own vertical AI in regulated industries, using agents-as-bundles (skills + connectors + sub-agents) to differentiate from raw API access and lock in enterprise workflows before competitors.
