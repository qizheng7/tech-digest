## Summary

Microsoft launched Agent 365 in general availability on May 1, 2026 — a $15/user/month management platform for enterprises to observe, govern, and secure AI agent fleets across Microsoft and third-party environments. The platform provides end-to-end visibility over agents with delegated credentials, discovers local and cloud-hosted agents (Windows, AWS Bedrock, Google Cloud), and enforces policy-based controls via Intune and Defender. Launch partners include Kore.ai, n8n, Genspark, Zendesk, and the Big Four consulting firms.

## Key Highlights

- Generally available at $15/user/month or included in Microsoft 365 E7
- Core capability: observe, govern, and secure agents operating with independent delegated credentials
- Discovers local agents (OpenClaw), Copilot Studio agents, and cloud agents across AWS Bedrock and Google Cloud
- Asset context mapping: devices, MCP servers, identities, and cloud resources per agent
- Windows 365 for Agents: secured cloud PC environments purpose-built for agentic workloads
- AWS Bedrock registry sync and Google Cloud connections in public preview — lifecycle governance (start/stop/delete) coming later
- Runtime blocking and malicious behavior detection in Microsoft Defender; Intune policy controls for local agents

## Why It Matters

As AI agents proliferate across enterprise stacks with delegated credentials and tool access, Agent 365 represents the first major platform play for the "agent control plane" category — a layer that will likely become mandatory for regulated industries.
