## Summary

OpenAI released GPT-5.5 and GPT-5.5 Pro on April 24, 2026, available across ChatGPT, Codex, and the API. The model is described as OpenAI's smartest and most intuitive to use yet, with exceptional gains in agentic coding, computer use, knowledge work, and early scientific research. Early adopters reported 6-hour autonomous task completions and hardware hacks that prior frontier models could not solve.

## Key Highlights

- GPT-5.5 API pricing: $5/M input tokens, $30/M output tokens; 1M context window
- GPT-5.5 Pro: $30/M input, $180/M output — targets complex multi-hour autonomous tasks
- Strongest gains in agentic coding, computer use, cross-tool task execution, and scientific research
- Available to Plus, Pro, Business, and Enterprise users in ChatGPT and Codex
- User reports: 6-hour autonomous data migrations with zero steering, hardware protocol reverse-engineering others couldn't crack
- Supports "/personality" slash command in Codex for customizing model tone during sessions

## Why It Matters

GPT-5.5 represents the clearest demonstration yet of truly long-running autonomous agents, with users reporting multi-hour unattended workflows — a meaningful step beyond single-turn assistance.
