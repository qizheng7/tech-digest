## Summary

Factory, maker of enterprise AI coding agents called "Droids," raised $150M Series C at a $1.5B valuation led by Khosla Ventures, with Sequoia, Insight Partners, and Blackstone participating. Founded in 2023 by Matan Grinberg and Eno Reyes, Factory's Droids handle the full software lifecycle — testing, code review, documentation, and deployment — not just code generation. Largest share of funding goes to long-horizon agent reliability research.

## Key Highlights

- $150M Series C at $1.5B valuation; led by Khosla Ventures; Keith Rabois joins board
- Investors: Sequoia, Insight Partners, Blackstone, Evantic Capital, 20VC, NEA, Mantis VC
- Product: "Droids" — AI agents covering coding, testing, code review, documentation, and deployment
- Founded 2023 by Matan Grinberg and Eno Reyes
- Research focus: long-horizon agent reliability — the hardest open problem in autonomous coding
- Expansion: London office opened Q2 2026; Tokyo planned Q3 2026; research team doubling

## Why It Matters

Factory's positioning as a full software-lifecycle orchestration layer rather than a coding assistant puts it in direct competition with GitHub Copilot and other narrower tools, betting that enterprise value is in end-to-end autonomy, not code completion.
