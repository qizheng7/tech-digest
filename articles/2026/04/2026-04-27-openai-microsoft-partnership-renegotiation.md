## Summary

OpenAI and Microsoft renegotiated their partnership on April 27, resolving a legal conflict created by OpenAI's $50B Amazon deal. Microsoft's exclusive API rights now expire in 2032, replacing an indefinite arrangement. OpenAI gains multi-cloud flexibility while Azure remains primary. Microsoft stops paying OpenAI revenue share; OpenAI continues paying Microsoft through 2030 with a new payment cap. Microsoft last quarter generated $7.5B from its OpenAI returns.

## Key Highlights

- Microsoft exclusivity expires 2032 (previously indefinite until AGI achievement)
- OpenAI can now use multiple cloud providers; Azure stays "primary" with first-launch rights
- Revenue sharing reversed: Microsoft stops paying OpenAI; OpenAI pays Microsoft through 2030 with a new cap
- Microsoft's last quarterly OpenAI returns: $7.5B
- Conflict originated from OpenAI's new Frontier agentic tool — both Amazon and Microsoft contracts claimed API exclusivity
- Microsoft retains its 27% ownership stake in OpenAI

## Why It Matters

The deal untangles a multi-cloud conflict that would have constrained OpenAI's growth and signals a maturing, more arms-length relationship between the two companies as OpenAI scales into a genuine enterprise cloud provider.
