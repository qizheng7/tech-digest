## Summary

Together AI launched day-0 access to NVIDIA Nemotron 3 Nano Omni, a unified multimodal model that processes video, images, audio, and text within a single reasoning framework. The model uses a hybrid Transformer-Mamba mixture-of-experts design with 30B total / ~3B active parameters, eliminating the need for fragmented multi-model pipelines for agentic workloads. Together AI's infrastructure delivers production-grade inference for the model on Hopper and Blackwell GPUs.

## Key Highlights

- **Architecture**: Hybrid Transformer-Mamba MoE — 30B total, ~3B active parameters per token
- **Multimodal**: Unified video, image, audio, and text reasoning in a single model; 256K token shared multimodal context window
- **Efficiency**: FP8 and NVFP4 quantization compatible; multi-token prediction generates multiple future tokens simultaneously
- **Use cases**: Customer service agents, financial analysts, computer use applications, and any agentic workload requiring deterministic multimodal reasoning
- **Infrastructure**: Together AI's research-optimized inference platform delivers faster multimodal reasoning with consistent latency at scale
- **Day-0 availability**: Model available the same day as NVIDIA's official release

## Why It Matters

Nemotron 3 Nano Omni's unified multimodal architecture removes architectural seams from agent pipelines, and Together AI's day-0 launch signals the maturing pattern of inference platforms racing to provide immediate access to frontier models as a competitive differentiator.
